# تعليمات استضافة بوت صحة للإجازات المرضية

## متطلبات النظام
- Python 3.8 أو أحدث
- pip (مدير حزم Python)

## خطوات الاستضافة

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
pip install hijri-converter
```

### 2. تكوين البوت
- تأكد من أن ملف `config_updated.py` يحتوي على:
  - `BOT_TOKEN`: توكن البوت من BotFather
  - `ADMIN_USER_ID`: معرف المستخدم المشرف

### 3. تشغيل البوت محلياً
```bash
python3 bot_updated.py
```

### 4. الاستضافة على منصات السحابة

#### Heroku
1. إنشاء ملف `Procfile`:
```
worker: python3 bot_updated.py
```

2. إنشاء ملف `runtime.txt`:
```
python-3.11.0
```

3. رفع المشروع إلى Heroku

#### Railway/Render
1. ربط المستودع بالمنصة
2. تعيين أمر البدء: `python3 bot_updated.py`
3. إضافة متغيرات البيئة إذا لزم الأمر

## الملفات المهمة
- `bot_updated.py`: الملف الرئيسي للبوت
- `pdf_generator_updated.py`: مولد ملفات PDF (محدث بالتنسيق الجديد)
- `config_updated.py`: ملف التكوين
- `requirements.txt`: قائمة المتطلبات
